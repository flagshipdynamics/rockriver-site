Frequently Asked Questions

Quick answers to the most common questions about RockWiFi Pro.

General

What is RockWiFi Pro?

RockWiFi Pro is a Windows desktop app for Wi-Fi site surveys, heatmaps, AP locating, and network diagnostics. Built for IT consultants, MSPs, and network admins who need professional-grade tools without enterprise pricing or cloud dependencies.

It is distributed through the Microsoft Store as a one-time purchase with no subscription. RockWiFi Pro is designed, built, and supported entirely in the United States by Rock River Research, LLC.

Is RockWiFi Pro free?

A free trial with full functionality is available so you can evaluate the product before purchasing. After the trial, RockWiFi Pro is a one-time purchase — no subscription, no account required, no recurring fees.

What operating systems are supported?

Windows 10 and Windows 11. RockWiFi Pro is available on the Microsoft Store.

Does RockWiFi Pro require an internet connection?

No. All features work offline. Wi-Fi scanning uses your local adapter. Map tiles are cached for offline use after first download. You can run full surveys, heatmaps, and diagnostics with no internet connectivity.

Does it require admin privileges?

No. Normal Wi-Fi scanning works without elevation. Some advanced features like raw packet capture would require admin, but the standard workflow — scanning, surveying, heatmaps, diagnostics — does not.

Wi-Fi Standards & Detection

What Wi-Fi standards does it detect?

RockWiFi Pro detects Wi-Fi 7 (802.11be), Wi-Fi 6/6E (802.11ax), Wi-Fi 5 (802.11ac), Wi-Fi 4 (802.11n), and legacy 802.11a/b/g. Standards are identified from 802.11 Information Elements in beacon and probe response frames.

Does it support the 6 GHz band?

Yes. RockWiFi Pro provides full 6 GHz spectrum visualization covering UNII-5 through UNII-8, channel recommendations, and detection of Wi-Fi 6E and Wi-Fi 7 access points operating on 6 GHz channels.

What is Multi-Link Operation (MLO)?

Multi-Link Operation (MLO) is a Wi-Fi 7 feature that allows simultaneous use of multiple bands — 2.4, 5, and 6 GHz — for a single connection. This improves throughput, reduces latency, and provides seamless failover. RockWiFi Pro detects and displays MLO capability on access points that support it.

What BSS metrics does it show?

RockWiFi Pro displays detailed per-BSS metrics including:

  • Channel width — 20, 40, 80, 160, or 320 MHz
  • Max PHY rate — the theoretical maximum data rate
  • Station count — number of associated clients
  • Channel utilization — airtime percentage
  • Beacon interval
  • Short GI — guard interval support
  • RSN security details — encryption and authentication suites
  • Country code
  • Wi-Fi standard — 4/5/6/6E/7
  • MLO status — Multi-Link Operation capability

Privacy & Data

Does it send data to the cloud?

No. All data stays on your machine. There is no telemetry, no cloud sync, and no connection to external servers. Floor plans, survey data, heatmaps, and reports never leave your laptop. See our Privacy Policy for details.

Can I export reports for clients?

Yes. RockWiFi Pro supports HTML and CSV export. HTML reports are self-contained, professionally formatted, and ready to attach to a ticket or email to a client. CSV export includes all BSS metrics for further analysis in a spreadsheet or reporting tool.

Comparison

How is this different from NetSpot or Ekahau?

RockWiFi Pro is a one-time purchase with no subscription. It includes both survey tools (heatmaps, AP locating, spectrum visualization) and network diagnostics (connection check, DNS compare, latency monitor, flight recorder) in a single application. No cloud account required. Wi-Fi 7 detection and 6 GHz band support are built in from day one.
