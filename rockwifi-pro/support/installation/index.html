Installing from the Microsoft Store

RockWiFi™ Pro Support

Home › RockWiFi Pro › Support › Installing from the Microsoft Store

Finding RockWiFi Pro

Open the Microsoft Store on Windows 10 or Windows 11 and search for RockWiFi Pro or Rock River Research. Click the app listing to view the product page.

Installation

Click Get or Install. The download is under 200 MB. No admin privileges are required — the Microsoft Store handles installation in your user context.

First Launch

Find RockWiFi Pro in your Start Menu. On first launch, the app opens to the Dashboard showing your current Wi-Fi connection status and signal strength.

Trial

The trial is full-featured — every tool works without restriction. Explore all features before purchasing. See Free Trial and Licensing for details.

Purchasing

When you're ready to subscribe, use the Purchase option within the app or on the Microsoft Store listing. It's a monthly subscription — cancel anytime.

Updates

Updates are delivered automatically through the Microsoft Store. You'll see available updates in the Store's Downloads & Updates section. See Updates and Support Policy for more information.

Uninstalling

Right-click the app in the Start Menu and select Uninstall, or go to Settings > Apps > Installed apps, find RockWiFi Pro, and uninstall. Survey data saved in your Documents folder is preserved unless you delete it manually.

Offline Installation

RockWiFi Pro requires the Microsoft Store for installation, but once installed, the app works fully offline. No internet connection is required for any feature except OpenStreetMap tile download.

System Requirements

  • Windows 10 (version 1903 or later) or Windows 11
  • 64-bit processor
  • 4 GB RAM minimum

See System Requirements for full details.
