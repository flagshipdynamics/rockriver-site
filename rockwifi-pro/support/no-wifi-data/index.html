No Wi-Fi Data Showing

RockWiFi Pro Support

Home › RockWiFi Pro › Support › No Wi-Fi Data Showing

If RockWiFi Pro is not displaying Wi-Fi scan data, signal information, or nearby access points, work through the steps below to identify and resolve the issue.

Check Your Wi-Fi Adapter

Make sure Wi-Fi is turned on. Some laptops have a hardware switch or Fn key combination that toggles the wireless radio. You can verify the adapter status in Windows Settings > Network & Internet > Wi-Fi.

Are You Connected to Wi-Fi?

The Dashboard requires an active Wi-Fi connection to show your current network's details. If you are connected via Ethernet only, the dashboard will appear blank. Connect to a Wi-Fi network first, then check the dashboard again.

Windows Wi-Fi Service

The WLAN AutoConfig service must be running for Wi-Fi scanning to work. To check:

  1. Open Services (press Win+R, type services.msc, press Enter).
  2. Find WLAN AutoConfig in the list.
  3. Ensure the status is Running and the startup type is set to Automatic.
  4. If it is stopped, right-click and select Start.

Adapter Compatibility

RockWiFi Pro works with any Windows-compatible Wi-Fi adapter. If your adapter does not appear in Device Manager under Network adapters, the driver may need to be updated or reinstalled. Check your laptop manufacturer's support site for the latest driver.

Scan Results Empty

If the Dashboard shows your connection details but UltraScan or the Congestion panel shows no APs, try these steps:

  1. Wait 5–10 seconds — the first scan after launch takes a moment to complete.
  2. Verify that airplane mode is off.
  3. Try disconnecting and reconnecting to your Wi-Fi network.
  4. Restart the app.

6 GHz Band Not Showing

Scanning the 6 GHz band requires both Windows 11 and a Wi-Fi 6E or Wi-Fi 7 adapter. If you do not have both, 6 GHz APs will not appear in scan results even if they are broadcasting nearby. This is a Windows platform limitation, not a RockWiFi Pro restriction.

Virtual Adapters

Some VPN software creates virtual Wi-Fi adapters that can interfere with scanning. If you see unexpected results or empty scans, check whether a VPN virtual adapter is active in your network settings and try disabling it temporarily.

Still Stuck?

If none of the above resolves the issue, contact support with the following information:

  • Your Windows version (e.g., Windows 11 23H2)
  • Your Wi-Fi adapter model (found in Device Manager under Network adapters)
  • A screenshot of the issue

You can also email support@rockriverresearch.com directly.
