RockWiFi™ Pro Features

Everything RockWiFi Pro ships, in detail.

RockWiFi Pro ships with 13 integrated tools for Wi-Fi surveying, network diagnostics, and reporting — including full Wi-Fi 7 Ready support with 802.11be detection, 6 GHz band analysis, and advanced BSS IE metrics. Everything below is shipping in v1.0.

Dashboard

The main screen and launch point for every tool. Shows a live Wi-Fi status card with your current SSID, signal strength in both percentage and dBm, frequency band, channel, and a scrolling real-time signal chart. Quick-launch buttons for all 13 tools are pinned to the dashboard so you never need to dig through menus.

Survey & Heatmaps

The headline feature. Import a floor plan image or use OpenStreetMap tiles for outdoor surveys. Walk the space and click to drop signal samples at your location on the map. RockWiFi Pro builds a color-coded heatmap using Kriging interpolation to fill in the gaps between sample points.

  • Multi-sample clustering — multiple samples at the same location are averaged for accuracy
  • Guided tour mode — follow a prompted path for consistent, repeatable coverage walks
  • Floor plan import — load PNG, JPG, or BMP floor plans and scale them to real-world dimensions
  • OpenStreetMap integration — survey outdoor areas using live map tiles without needing a floor plan
  • Heatmap export — save the completed heatmap as part of an HTML report for client deliverables

AP Locator

Physically locate any access point in the building. A hot/cold signal gauge shows real-time signal strength as you walk toward (or away from) the target AP. Pin positions on the floor plan and use trilateration to narrow down the location.

  • Signal gauge — real-time hot/cold indicator that responds as you move
  • Trilateration — pin 3+ positions with known signal readings to triangulate the AP's location on the floor plan
  • Channel congestion chart — see which channels are crowded and which are clear at the current location

Connection Check

A full-stack connectivity diagnostic that tests each layer in sequence: Wi-Fi adapter status, gateway reachability, internet connectivity, DNS resolution, latency, jitter, and packet loss. One click tells you exactly where the connection breaks — is it the adapter, the gateway, DNS, or the internet?

Target Test

Probe a specific host or port using multiple methods:

  • Ping — ICMP echo with round-trip time
  • TCP connect — test whether a specific port is open and responding
  • HTTP — send an HTTP request and inspect the response code, headers, and timing
  • Traceroute — map the path to the target and identify where latency or loss occurs
  • TLS certificate inspection — view certificate details, expiration, chain validity, and cipher suite

DNS Compare

Query multiple DNS resolvers simultaneously and compare the results side-by-side. Built-in presets for system resolver, Google (8.8.8.8), Cloudflare (1.1.1.1), and Quad9 (9.9.9.9), plus custom resolver support. Spot DNS filtering, hijacking, propagation delays, or stale records without leaving the app.

Latency Monitor

Continuous ping to one or more targets with a live scrolling chart. Tracks round-trip time, jitter, and packet loss over time with min/max/average and percentile statistics (P50, P95, P99). Leave it running during a meeting, a firmware upgrade, or overnight — come back with a complete timeline of network behavior.

Flight Recorder

Background logging of connectivity metrics for post-incident analysis. Flight Recorder captures latency, packet loss, DNS resolution times, and connection state changes at configurable intervals. When something goes wrong at 2 AM, you have the data to prove what happened and when.

Reports & History

Browse past diagnostic results, survey data, and latency logs. Export any result as an HTML report or plain text. HTML reports are self-contained, professional, and readable in any browser — attach them to tickets, email them to clients, or archive them for compliance.

Wi-Fi 7 Ready

RockWiFi Pro detects and reports on 802.11be (Wi-Fi 7) access points as they appear in your environment. Whether you are planning a Wi-Fi 7 migration or auditing an existing deployment, every panel in the application surfaces Wi-Fi 7 capabilities automatically.

  • 320 MHz channel width detection — identifies APs operating at the full 320 MHz width introduced by Wi-Fi 7
  • Multi-Link Operation (MLO) awareness — detects and flags APs advertising MLO capability for simultaneous multi-band connections
  • 6 GHz band support — full coverage of the 6 GHz spectrum including UNII-5, UNII-6, UNII-7, and UNII-8 bands
  • 4096-QAM recognition — identifies APs capable of 4096-QAM modulation for maximum throughput
  • Wi-Fi 7 readiness scoring — professional reports include a Wi-Fi 7 readiness score based on current AP capabilities, band availability, and client support
  • Upgrade path recommendations — actionable findings that identify which APs should be upgraded and what Wi-Fi 7 benefits the environment would gain

802.11 Standard Detection

RockWiFi Pro parses BSS Information Elements (IEs) from every visible access point to determine the exact 802.11 standard in use. Instead of raw protocol identifiers, every panel displays friendly names — Wi-Fi 7 (802.11be), Wi-Fi 6E (802.11ax on 6 GHz), Wi-Fi 6 (802.11ax), Wi-Fi 5 (802.11ac), Wi-Fi 4 (802.11n), and legacy 802.11a/b/g. Standard detection is consistent across the Dashboard, AP Locator, UltraScan, Channel Congestion, and all report outputs, so you always know what generation of hardware is on your network.

Channel Congestion Analysis

A dedicated spectrum visualization tool that shows every access point on every channel across all three bands. Identify congested channels, find clear spectrum, and make informed channel planning decisions.

  • Tri-band spectrum charts — separate visualizations for 2.4 GHz, 5 GHz, and 6 GHz bands on a single screen
  • UNII band labeling — 5 GHz and 6 GHz charts are annotated with UNII-1 through UNII-8 band boundaries for regulatory clarity
  • Per-channel congestion scoring — each channel receives a congestion score based on co-channel AP count, adjacent-channel overlap, and aggregate signal levels
  • Channel recommendations — automatic best-channel suggestions for each band based on current congestion analysis
  • AP detail cards — click any AP on the chart to see its 802.11 standard, channel width, connected client count, airtime utilization, and security configuration

UltraScan

A continuous background AP scanner that tracks every visible access point over time. UltraScan runs passively alongside your other work and builds a complete picture of the RF environment without requiring dedicated survey walks.

  • Persistent AP tracking — maintains a live table of all detected APs with first-seen and last-seen timestamps
  • 10 sort modes — sort by signal strength, 802.11 standard, channel width, max PHY rate, channel, SSID, security, band, client count, or last seen
  • Per-AP signal history — select any AP to view its signal strength over time with stability metrics and trend analysis (improving, degrading, or stable)
  • CSV and HTML export — export the full scan results with complete BSS data including all IE-derived metrics
  • Session save and load — save a scan session to disk and reload it later for comparison or continued monitoring

Advanced BSS Metrics

Every panel in RockWiFi Pro surfaces detailed BSS (Basic Service Set) metrics parsed directly from 802.11 Information Elements. These metrics appear in the Dashboard, AP Locator, UltraScan, Channel Congestion, and all report outputs:

  • Channel width — 20, 40, 80, 160, or 320 MHz as reported by the AP
  • Max PHY rate — the maximum physical-layer data rate in Mbps
  • Station count — the number of currently connected clients (from BSS Load IE)
  • Channel utilization — airtime percentage consumed on the channel (from BSS Load IE)
  • Beacon interval — the AP's beacon transmission interval in milliseconds
  • Short guard interval — whether the AP supports short GI for higher throughput
  • RSN security detail — full security breakdown including AKM suite (PSK, SAE, 802.1X), pairwise cipher (CCMP, GCMP-256), and group cipher
  • Country code — the regulatory domain advertised by the AP

Professional Reports

Generate detailed, client-ready reports that go beyond raw data. The reporting engine analyzes your survey and scan results to produce scored assessments with actionable findings.

  • Five scored categories — coverage quality, channel congestion, signal-to-noise ratio (SNR), roaming readiness, and overall network health, each on a 0–100 scale
  • Weighted composite score — an overall network health score calculated from weighted category scores for a single summary metric
  • Findings engine — automatically generated findings with severity ratings (critical, warning, informational) based on detected issues
  • Capability analysis — warnings for legacy 802.11 standards still in use, channel width recommendations, WPA3 upgrade suggestions, and Wi-Fi 7 readiness assessment
  • CSV sample data export — export raw sample data for further analysis in spreadsheets or other tools

Application Design

Local-First

All data — surveys, heatmaps, diagnostic results, settings — is stored locally on your machine. Nothing is transmitted to Rock River Research or any third-party service. No cloud account, no sign-up, no sync.

Light and Dark Themes

Toggle between light and dark UI themes. Dark theme (background #1a1a1a, accent #0078d4) is the default. Changes apply immediately with no restart required.

Windows Native

Built with Python and PyQt6 for responsive, native-feeling performance on Windows. Not a web wrapper.

Microsoft Store Distribution

Clean install from the Microsoft Store. Automatic updates. Free trial with full functionality. Monthly subscription — cancel anytime.
