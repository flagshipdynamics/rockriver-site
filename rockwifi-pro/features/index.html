RockWiFi Pro Features

Everything RockWiFi Pro ships, in detail.

RockWiFi Pro ships with 9 integrated tools for Wi-Fi surveying, network diagnostics, and reporting. Everything below is shipping in v1.0.

Dashboard

The main screen and launch point for every tool. Shows a live Wi-Fi status card with your current SSID, signal strength in both percentage and dBm, frequency band, channel, and a scrolling real-time signal chart. Quick-launch buttons for all 9 tools are pinned to the dashboard so you never need to dig through menus.

Survey & Heatmaps

The headline feature. Import a floor plan image or use OpenStreetMap tiles for outdoor surveys. Walk the space and click to drop signal samples at your location on the map. RockWiFi Pro builds a color-coded heatmap using Kriging interpolation to fill in the gaps between sample points.

  • Multi-sample clustering — multiple samples at the same location are averaged for accuracy
  • Guided tour mode — follow a prompted path for consistent, repeatable coverage walks
  • Floor plan import — load PNG, JPG, or BMP floor plans and scale them to real-world dimensions
  • OpenStreetMap integration — survey outdoor areas using live map tiles without needing a floor plan
  • Heatmap export — save the completed heatmap as part of an HTML report for client deliverables

AP Locator

Physically locate any access point in the building. A hot/cold signal gauge shows real-time signal strength as you walk toward (or away from) the target AP. Pin positions on the floor plan and use trilateration to narrow down the location.

  • Signal gauge — real-time hot/cold indicator that responds as you move
  • Trilateration — pin 3+ positions with known signal readings to triangulate the AP's location on the floor plan
  • Channel congestion chart — see which channels are crowded and which are clear at the current location

Connection Check

A full-stack connectivity diagnostic that tests each layer in sequence: Wi-Fi adapter status, gateway reachability, internet connectivity, DNS resolution, latency, jitter, and packet loss. One click tells you exactly where the connection breaks — is it the adapter, the gateway, DNS, or the internet?

Target Test

Probe a specific host or port using multiple methods:

  • Ping — ICMP echo with round-trip time
  • TCP connect — test whether a specific port is open and responding
  • HTTP — send an HTTP request and inspect the response code, headers, and timing
  • Traceroute — map the path to the target and identify where latency or loss occurs
  • TLS certificate inspection — view certificate details, expiration, chain validity, and cipher suite

DNS Compare

Query multiple DNS resolvers simultaneously and compare the results side-by-side. Built-in presets for system resolver, Google (8.8.8.8), Cloudflare (1.1.1.1), and Quad9 (9.9.9.9), plus custom resolver support. Spot DNS filtering, hijacking, propagation delays, or stale records without leaving the app.

Latency Monitor

Continuous ping to one or more targets with a live scrolling chart. Tracks round-trip time, jitter, and packet loss over time with min/max/average and percentile statistics (P50, P95, P99). Leave it running during a meeting, a firmware upgrade, or overnight — come back with a complete timeline of network behavior.

Flight Recorder

Background logging of connectivity metrics for post-incident analysis. Flight Recorder captures latency, packet loss, DNS resolution times, and connection state changes at configurable intervals. When something goes wrong at 2 AM, you have the data to prove what happened and when.

Reports & History

Browse past diagnostic results, survey data, and latency logs. Export any result as an HTML report or plain text. HTML reports are self-contained, professional, and readable in any browser — attach them to tickets, email them to clients, or archive them for compliance.

Application Design

Local-First

All data — surveys, heatmaps, diagnostic results, settings — is stored locally on your machine. Nothing is transmitted to Rock River Research or any third-party service. No cloud account, no sign-up, no sync.

Light and Dark Themes

Toggle between light and dark UI themes. Dark theme (background #1a1a1a, accent #0078d4) is the default. Changes apply immediately with no restart required.

Windows Native

Built with Python and PyQt6 for responsive, native-feeling performance on Windows. Not a web wrapper.

Microsoft Store Distribution

Clean install from the Microsoft Store. Automatic updates. Free trial with full functionality. One-time purchase — no subscription.
