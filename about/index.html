Who We Are

Rock River Research is a software company building professional-grade tools across network operations and personal finance.

We started with a straightforward observation: the tools that professionals depend on every day are often bloated, overengineered, or built by people who don’t actually use them. We take a different approach. We build software we use ourselves, we ship when it’s ready, and we iterate based on how the tools perform in real workflows — not on feature checklists.

Our Products

Network Operations

RockWiFi Pro™ is a professional Wi-Fi survey, analysis, and network diagnostics suite. Floor-plan heatmaps, AP locating, flight recording, and client-ready reports. Built for IT consultants, MSPs, and network admins who need professional Wi-Fi analysis without the enterprise price tag.

RockWiFi Essentials™ is the core Wi-Fi scanning, channel congestion analysis, and network diagnostics toolkit. UltraScan, Connection Check, Target Test, DNS Compare, and Latency Monitor — the tools every tech needs on their laptop.

RockTerm™ is a Windows-native terminal emulator with SSH, Telnet, serial console, RDP, file transfer, scripting, and opt-in AI assistance. 110+ features built for network engineers.

Personal Finance & Trading

RockSolid Money™ is a local-first personal finance app — a modern replacement for Microsoft Money and Quicken — that imports from any bank and tracks accounts, budgets, cash flow, investments, and net worth. It comes in Essentials and Pro editions. RockSolid Trader™ is a local-first trading platform. Both are currently in development.
