AI API Key Setup

Last updated: April 2026

RockTerm's AI features are powered by the Anthropic API. To use them, you need to provide your own Anthropic API key. This article walks you through obtaining a key, entering it in RockTerm, and understanding the associated costs.

Getting an Anthropic API Key

  1. Visit console.anthropic.com in your web browser.
  2. Create an Anthropic account if you do not already have one, or sign in to your existing account.
  3. Navigate to the API Keys section of the console.
  4. Click Create Key to generate a new API key.
  5. Copy the key immediately. Anthropic will only display the full key once; if you lose it, you will need to generate a new one.

The Anthropic API is a paid service with usage-based pricing. RockTerm does not charge separately for AI features — the only cost is your direct API usage with Anthropic. You can review Anthropic's current pricing on their website.

Entering the Key in RockTerm

  1. Open RockTerm and go to Settings (via the menu bar or the keyboard shortcut).
  2. Select the AI tab in the settings panel.
  3. Locate the Anthropic API Key field.
  4. Paste your API key into the field and click Save.
  5. AI features will become available immediately. You do not need to restart the application.

API Key Security

Your API key is sensitive information and should be treated like a password. RockTerm handles your key with the following safeguards:

  • Stored locally. Your API key is stored on your local machine only. It is never transmitted to Rock River Research or any third party.
  • Transmitted only to Anthropic. The key is sent exclusively to Anthropic's API endpoint (api.anthropic.com) over HTTPS when you invoke an AI feature.
  • Never logged or cached remotely. RockTerm does not log your API key to any file, crash report, or analytics service.

As a best practice, treat your API key like a password. Do not share it with others, do not paste it into untrusted applications, and regenerate it promptly if you believe it has been compromised.

Usage and Cost

AI queries consume tokens through the Anthropic API. Here is what you should know about costs:

  • Lightweight usage. Normal, day-to-day use of RockTerm's AI features is very lightweight. Most users can expect costs of just a few cents per day under typical usage patterns.
  • Token consumption. Each AI query sends a prompt and receives a response, both of which consume tokens. Longer terminal context or more complex questions will use more tokens.
  • Monitor your usage. You can view your current API usage and spending at any time by visiting the Anthropic console.
  • Set spending limits. Anthropic allows you to configure monthly spending limits in the console. We recommend setting a limit that aligns with your budget to avoid unexpected charges.
