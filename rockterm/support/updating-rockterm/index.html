Automatic Updates via Microsoft Store

RockTerm is distributed through the Microsoft Store, which handles updates automatically. By default, the Microsoft Store downloads and installs app updates in the background when your system is connected to the internet and idle. You do not need to take any action to stay on the latest version.

Automatic updates are applied when RockTerm is not running. If RockTerm is open when an update becomes available, the update will be applied the next time you close and reopen the application.

To verify that automatic updates are enabled in the Microsoft Store:

  1. Open the Microsoft Store.
  2. Click your profile icon in the top-right corner and select App settings (or Settings).
  3. Ensure App updates is toggled On.

Checking for Updates Manually

If you want to check for and install an update immediately rather than waiting for the automatic cycle:

  1. Open the Microsoft Store.
  2. Click Library in the bottom-left corner (on Windows 11) or the three-dot menu and select Downloads and updates (on Windows 10).
  3. Click Get updates in the top-right corner.
  4. The Store will check for updates for all installed apps. If an update is available for RockTerm, it will begin downloading immediately.
  5. Once the update is downloaded and installed, close and reopen RockTerm to use the new version.

You can verify your current version by opening RockTerm and navigating to Settings > About. The version number is displayed at the top of the About page.

Changelog

A detailed changelog documenting new features, improvements, bug fixes, and known issues for each release is available at /rockterm/changelog/. We recommend reviewing the changelog after major updates to learn about new features and any changes that may affect your workflow.

You can also access the changelog directly from within RockTerm via Settings > About > What's New.

Settings Preserved After Updates

Updates do not modify or reset your user data. The following are preserved across all updates:

  • Saved connections and connection folders
  • Application settings and preferences
  • Custom keyboard shortcuts
  • Color scheme customizations
  • AI configuration (API key, model preferences)
  • Session logs

In rare cases where an update introduces a new settings format, RockTerm automatically migrates your existing settings to the new format on first launch after the update. A backup of the previous settings is created automatically. If migration fails for any reason, the affected settings revert to their defaults and a notification is shown explaining what happened.

Troubleshooting Update Issues

If the Microsoft Store is not updating RockTerm, try the following:

  • Check your internet connection. Updates require an active connection to the Microsoft Store servers.
  • Clear the Store cache. Press Win+R, type wsreset.exe, and press Enter. This resets the Store cache and can resolve stuck downloads.
  • Check for pending Windows updates. Some Store functionality depends on Windows being up to date. Install any pending updates via Settings > Windows Update.
  • Check disk space. Ensure sufficient free space on your system drive for the update to download and install.
  • Restart and retry. Restart your computer, open the Microsoft Store, and try Get updates again.
