Getting Started with RockTerm

Step 1: Install RockTerm

Download RockTerm from the Microsoft Store. The free trial includes full functionality so you can evaluate the product before purchasing. Once installed, launch RockTerm from your Start menu or taskbar.

Step 2: Create a Connection

Open the connection manager from the sidebar or main menu. Click New Connection and choose your connection type:

SSH

  • Host — the hostname or IP address of the remote machine
  • Port — the SSH port (default: 22)
  • Username — your login username
  • Authentication — choose password or SSH key, and provide the key file path if applicable

Telnet

  • Host — the hostname or IP address of the target device
  • Port — the Telnet port (default: 23)

Serial Console

  • COM Port — select the serial port (e.g., COM3)
  • Baud Rate — the connection speed (common values: 9600, 115200)
  • Data Bits / Stop Bits / Parity — configure as required by your device (defaults: 8, 1, None)

Save the profile. You can organize connections into groups for different environments or projects.

Step 3: Connect

Select your saved connection from the connection manager and click Connect. RockTerm opens a new terminal tab with your session. You can open multiple connections in separate tabs and switch between them — mixing SSH, Telnet, and serial sessions as needed.

Step 4: Using AI Assistance

With an active terminal session, invoke the AI assistant using the AI button in the toolbar or the configured keyboard shortcut. The assistant can see your recent terminal output and provide contextual help. Ask it about error messages, command syntax, configuration questions, or have it explain complex output.

Tips

  • Organize SSH, Telnet, and serial connections into groups for different environments (production, staging, lab).
  • Customize your terminal font and colors in Settings.
  • AI features require an active internet connection for the AI provider API.
