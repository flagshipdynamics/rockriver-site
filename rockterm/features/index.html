Features

AI Integration

RockTerm's AI assistant lives inside your terminal session. When you need help, invoke the assistant and it reads your recent terminal output to provide contextual guidance. It can interpret error messages, suggest commands, explain complex output like routing tables or interface configurations, and help you work through unfamiliar log formats.

AI features are entirely opt-in. Nothing is sent to any external AI provider unless you explicitly trigger the assistant. When you do, only the relevant terminal context is transmitted — and you can see exactly what's being sent. The AI understands networking context: BGP neighbor tables, OSPF adjacencies, firewall rules, interface stats, and more.

SSH Terminal

RockTerm provides fast, reliable terminal emulation with full support for standard SSH connections. Authenticate with passwords or SSH keys, configure terminal settings like font, colors, and scrollback buffer size, and run multiple simultaneous sessions in tabs. The terminal renders quickly and handles high-throughput output without lag.

Connection Manager

Save connection profiles with host, port, username, and key file path. Organize profiles into groups — by environment, project, client, or however you work. Quick-connect to any saved host with a single click. Import and export connection lists for backup or sharing across machines.

Interface

A dark theme optimized for extended terminal use. Resizable panes let you view multiple sessions side by side. Customize your font family, font size, and color scheme. The interface is keyboard-driven with configurable shortcuts, so you can stay in your workflow without reaching for the mouse.

Microsoft Store Integration

RockTerm is installed directly from the Microsoft Store. Updates are delivered automatically. A free trial is available so you can evaluate the full product before purchasing. When you're ready, it's a one-time purchase — no subscription, no recurring charges.
