Built for people who manage real infrastructure.

If you spend your day SSH'd into routers, configuring switches over serial, or managing fleets of Linux servers, your terminal emulator matters. RockTerm was designed from the ground up for that reality — not as a general-purpose shell, but as a purpose-built tool for network engineers and sysadmins who need reliable multi-protocol access and efficient connection management.

What RockTerm Does

🔒

SSH, Telnet, and Serial Console

All three protocols in one application. SSH with key-based auth and agent forwarding. Telnet for legacy device access. Serial console with configurable baud rate, data bits, parity, and flow control. No need to juggle separate tools.

⌨

AI-Powered Assistance

Invoke the AI assistant mid-session to interpret error messages, explain routing tables, suggest configuration syntax, or work through unfamiliar log output. The assistant reads your terminal context so you don't have to copy-paste into a browser. Entirely opt-in — uses your own Anthropic API key.

☰

Connection Manager

Save and organize connection profiles across all three protocols. Group by environment, client, project, or role. Connect with one click. Import from existing tools and export for backup or team sharing.

⚡

Native Windows Performance

Built with native UI toolkits — not Electron, not a web view. RockTerm launches fast, renders terminal output without lag, and stays responsive even with high-throughput sessions. Designed for long working sessions.

▦

Professional Interface

Dark theme optimized for extended use. Tabbed sessions with keyboard-driven navigation. Customizable fonts and colors. Resizable layout. Built for people who spend hours in the terminal, not minutes.

⇩

Microsoft Store Distribution

Clean install from the Microsoft Store. Automatic updates. Free trial with full functionality so you can evaluate before purchasing. One-time purchase — no recurring charges.

How People Use RockTerm

Network Operations

SSH into routers and switches, console into new hardware over serial, and manage Telnet sessions for legacy devices — all from one window. Group connections by site or environment. Use AI to quickly interpret show commands, log output, and error messages.

System Administration

Manage fleets of Linux and Windows servers across tabbed SSH sessions. Jump between production, staging, and development environments with saved connection profiles. Let the AI assistant help debug service failures or parse unfamiliar log formats.

Infrastructure Lab Work

Console into lab equipment over serial while maintaining SSH sessions to production. Keep all your connections organized without switching between PuTTY, a serial terminal, and a Telnet client. Import existing PuTTY sessions to get started quickly.

What You're Getting
