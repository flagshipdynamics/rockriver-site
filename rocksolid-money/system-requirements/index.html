System Requirements

What you need to run RockSolid Money.

Home › RockSolid Money › System Requirements

RockSolid Money is in development. The requirements below reflect the current build and will be confirmed at release.

Requirement Details
Operating System Windows 10 (version 1903 or later) or Windows 11
Architecture x64 (64-bit)
RAM 4 GB minimum, 8 GB recommended for large transaction histories and investment portfolios
Disk Space 300 MB for installation, plus space for your ledger, imported originals, and any attached documents
Display 1280x720 minimum, 1920x1080 recommended
Network Not required. The application does not connect to your bank and stores nothing in the cloud. An internet connection is needed only to download the app and any optional price updates.
Bank Connection None. You import files you download yourself — no online banking credentials are requested or stored.
Distribution Microsoft Store (Windows 10/11)
Admin Rights Not required. A standard user account is sufficient.

Supported Import Formats

  • CSV / TSV — exports from banks, credit cards, and brokerages. Profiles for major US institutions ship with the app; unrecognized layouts can be mapped once and saved.
  • OFX / QFX — the common download format offered by most US banks and brokerages.
  • QIF — the usual export path out of Microsoft Money and Quicken.

Data Location

  • Your ledger and imported originals are stored in a file under your Windows user profile.
  • Nothing is written to a cloud service, and no account is required to use the application.
  • The data file can be backed up and restored on demand.

Editions

  • Essentials and Pro share the same requirements — they are one application built from a single codebase.
  • See the edition comparison for what each includes.
