Import Evidence

RockSolid Money™ Support

Home › RockSolid Money › Support › Import Evidence

A ledger you trust is one that can show its work. Every transaction that arrived through an import keeps a link to the exact row of the exact file it came from, and the interpretation that turned that row into a ledger entry.

Viewing evidence

Select a transaction (in the register, All Transactions, or anywhere else) and choose View Import Evidence in the Inspector. You'll see:

  • The raw source row as the bank wrote it,
  • which import profile interpreted it and how each column was read,
  • the import session it arrived in, with a link to the full session.

Why this matters

When a number looks wrong, the question is always “did the bank say that, or did the import misread it?” Evidence answers it in one click — no hunting through downloaded files. If the interpretation was wrong, roll back the session and correct the profile (see Import History and Rollback); if the bank's row was wrong, you know to take it up with the bank.

Where the files live

Imported source files are stored locally alongside your ledger, on your machine, like everything else (see Privacy and Your Data).
