Keyboard Shortcuts

RockSolid Money™ Support

Home › RockSolid Money › Support › Keyboard Shortcuts

The shortcuts that matter, all of them:

ShortcutAction
Ctrl+IImport a file
Ctrl+F or Ctrl+KGlobal search
Ctrl+HImport History
Ctrl+CCopy the selected table rows (paste into a spreadsheet)
Ctrl+,Settings
F1User guide
Ctrl+QExit

Menus list their shortcuts beside each action, so anything not on this list is discoverable where you'd use it.
