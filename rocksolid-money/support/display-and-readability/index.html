Display and Readability

RockSolid Money™ Support

Home › RockSolid Money › Support › Display and Readability

RockSolid Money is built to be read for an hour at a time without strain, in the theme you prefer.

Themes

View > Theme: Dark, Light, or System (follow Windows). Every screen, chart, and dialog follows the choice.

Bold Mode

View > Bold Mode makes text across the application larger and genuinely bold — menus, tables, cards, charts. It's built for high-DPI displays, larger Windows text scaling, and anyone who'd rather not lean in. Toggle it off and the regular weights return immediately.

The layout adapts

  • The navigation rail collapses to an icons-only strip (« / ») when you want the room.
  • The right-hand Inspector can be hidden (View > Inspector) and reappears when you select something that has detail to show.
  • Dashboard panels reflow as the window narrows instead of crushing their contents.

Money is typeset to be compared

Amounts align in columns with consistent decimal places, and color is used sparingly on purpose: an ordinary expense is not painted red, and a column of balances is not painted green. Strong red is reserved for things that are actually wrong — an overdraft, a failed backup, an overdue bill — so that when you see it, it means something.
