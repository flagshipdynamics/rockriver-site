Multiple Financial Files (Profiles)

RockSolid Money™ Support

Home › RockSolid Money › Support › Multiple Financial Files (Profiles)

One installation can keep several completely separate financial files — profiles. Your personal file, a parent's finances you help manage, a small side venture: each is its own ledger, its own backups, its own everything.

Switching

Open the file selector (your household name, top left) > Switch Profile. The application restarts into the chosen file — deliberately, so one file's numbers can never appear under another file's name. The file you were in is saved; a normal launch reopens the profile you last used.

Creating a new profile

New Profile… names a fresh, empty file and opens it — which lands on the setup interview, like any new file.

The sample household

A built-in Demo — Sample Data profile contains a fully populated fictional household — two years of transactions, bills, budgets, goals, investments, crypto, the works. It's the fastest way to explore every screen before importing anything real, and it never touches your own file. It only opens when you explicitly choose it.

Renaming

Rename This File… changes the display name of the current profile — a label, not a filename; nothing moves on disk.
